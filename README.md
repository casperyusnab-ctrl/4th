# CSS Units, Box Model and Fonts Task

This project is a basic HTML and CSS practice task.
The goal of this task is to understand CSS concepts like **units (vh, vw), box model, and basic styling**.

In this task, an image is displayed inside a div and a button is added below it.
CSS is used to control spacing, layout, and styling without using flexbox, positioning, or overflow.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Make sure **index.html** and **style.css** are in the same folder.
4. Open **index.html** in any web browser.
5. The styled webpage will appear as shown in the output.
